# django-crud-todo
A simple and beginner-friendly To-Do web application built using Django. This project allows users to manage their daily tasks with full CRUD (Create, Read, Update, Delete) functionality.

Features

* Add new tasks
* View all tasks
* Update/edit existing tasks
* Delete tasks
* Simple and clean user interface

Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS
* Database: SQLite (default Django database)

How It Works

* Tasks are stored in the database using Django models
* Views handle the business logic and user requests
* Templates display data and interact with users
* URL routing connects different pages of the app

Key Concepts Used

* Django MVT Architecture (Model, View, Template)
* Django ORM for database operations
* Form handling and validation
* URL mapping and request-response cycle
